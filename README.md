| [dotenv](https://www.npmjs.com/package/dotenv)                            | `npm i dotenv`      | ``` import dotenv from 'dotenv'; dotenv.config(); ```                                                      | 3rd-party | Lädt Umgebungsvariablen aus einer .env-Datei in process.env                                                                                                                                                                                |
